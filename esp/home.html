VerifiQ
Verificador de originalidad de sneakers con certificado en blockchain

Este proyecto es un servicio donde el cliente sube algunos retratos de su sneakers y, después de pagar una tasa, recibe un certificado en blockchain que comprova la originalidad de su producto. Este proyecto tuvo un desarrollo complejo por integrar blockchain, backend e frontend. Yo desarrollé el frontend, donde, utilizando HTML, CSS e Javascript, entregué a tiempo un buen resultado final:

En este proyecto, adquirí conocimiento satisfactorio en HTML y CSS, aprendí a realizar animaciones y aceptar el JS cómo una solución de problemas mágico. El Javascript nos ayudó en el login, haciendo una animación básica, eliminando la opción "login" después del usuario logarse. Mi equipo logró una buena colaboración y comunicación através del Github. Ese proyecto fue muy importante para mi aprendizaje acerca de los conceptos importantes del Frontend, con trabajo en equipo con la finalidad del desarrollo ideal de la aplicación.

Juego de plataformas básico

Usando principalmente Javascript, creé un "juego de plataformas" donde los bloques se mueven hacia el jugador, quien simplemente salta. La mayor dificultad fue crear la colisión y el HUD, que inciden directamente en la jugabilidad, como finalizar el juego con un botón y cambiar el volumen (usando una etiqueta). Fue un proyecto interesante, donde agregué algunos sonidos, como saltar, ganar, perder y sumar puntos.

Puse un límite de 50 puntos para que no fuera infinito, utilice los colores temáticos del portafolio y todos los sonidos utilizados fueron de la franquicia de Mario.

Estudos acerca de lazy loading

Neste proyecto yo crié una página de imágenes de alta resolución y, utilizando las DevTools del Google Chrome, simulé una conexión 4G lenta, para mostrar la implementación del lazy loading.

Este setup demonstra cómo el lazy loading genera un delay, disminuyendo el tiempo de cargamento inicial y ahorrando internet para conexiones lentas.
